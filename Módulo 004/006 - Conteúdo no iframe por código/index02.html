Conteúdo no iframe por código

Podemos usar o parâmetro 'srcdoc' dos iframes para inserir códigos/tags html para criarmos uma página padrão que será exibida primeiro antes de iniciarmos nossa navegação pelos iframes.

  • Página vermelha
  • Página azul
  • Página amarela

Caso sua página padrão for muito complexa é mais recomendável criarmos um novo documento html!
